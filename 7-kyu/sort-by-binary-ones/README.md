<h2><a href=https://www.codewars.com/kata/59eb28fb0a2bffafbb0000d6/train/javascript target="_blank">Sort by binary ones</a></h2><h3>7 kyu</h3><p>In this example, you need to implement a function that sorts a list of integers based on their binary representation. </p><p>The rules are simple:</p><ol><li>sort the list based on the amount of <code>1</code>s in the binary representation of each number. </li><li>if two numbers have the same amount of <code>1</code>s, the shorter string goes first. (ex: "11" goes before "101" when sorting 3 and 5 respectively)</li><li>if the strings have the same length, lower decimal number goes first. (ex:  21 = "10101" and 25 = "11001", then 21 goes first as it's lower)</li></ol><p>Examples:</p><ul><li><p>Input: <strong>[1,15,5,7,3]</strong> </p><ul><li>( in binary strings is: <code>["1", "1111", "101", "111", "11"]</code>)</li></ul></li><li><p>Output: <strong>[15, 7, 3, 5, 1]</strong></p><ul><li>(and after <em>sortByBinaryOnes</em> is: <code>["1111", "111", "11", "101", "1"]</code>)</li></ul></li></ul>
